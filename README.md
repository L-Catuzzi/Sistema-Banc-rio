Resumo projeto
Criar um sistema bancário utilizando classes em python e conceitos de programação orientada a objeto

## Funcionalidades
Cliente(base):
    - Descrição do cliente v

Clientes(gold):
    - Compra Crédito

Clientes(black):
    - Investir 

Conta: 
    - Depositar v
    - Sacar v
    - checar saldo v
    - transferir v

Funcionário:
    - Checar dados do cliente v
    - Checar dados da conta v
    - Promover conta  v
    - Deletar conta v 
    - Deletar Cliente v
    - Criar conta v
    - Listar transações v

Utilidades:
    - Encontrar cliente v
    - Encontrar conta v

Testes
- Buscar cliente que não existe
- Transação sem saldo
- Solicitar crédito
- Busca de Valores inválidos
